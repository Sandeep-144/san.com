Cyber Security

Module 1: Systems Vulnerability Scanning

This module covers the essential concepts and tools used in vulnerability scanning, including:

  • Overview of vulnerability scanning: understanding its significance in identifying weaknesses.
  • Open Port / Service Identification: techniques for detecting open ports and services on a network.
  • Banner / Version Check: methods for identifying service versions through banner grabbing.
  • Traffic Probe: analyzing traffic to uncover potential vulnerabilities.
  • Vulnerability Probe: tools and techniques for probing systems for known vulnerabilities.
  • Examples of vulnerabilities and tools such as OpenVAS and Metasploit for vulnerability assessment.
  • Networks Vulnerability Scanning: using tools like Netcat and Socat to identify vulnerabilities.
  • Understanding Port and Service tools: analyzing tools like Datapipe, Fpipe, and WinRelay.
  • Network Reconnaissance: utilizing Nmap and THC-Amap for comprehensive network scanning.
  • Network Sniffers and Injection tools: practical usage of Tcpdump, Windump, Wireshark, Ettercap, Hping, and Kismet.

Module 2: Network Defense Tools

This module focuses on essential network defense mechanisms, including:

  • Firewalls and Packet Filters: understanding the basics of firewalls and their role in network security.
  • Differences between Packet Filter and Firewall: analyzing the characteristics and functionality.
  • Packet Characteristics to Filter: identifying which characteristics are essential for filtering.
  • Stateless vs. Stateful Firewalls: understanding the differences and use cases for each type.
  • Network Address Translation (NAT) and Port Forwarding: exploring techniques to enhance security and connectivity.
  • Introduction to Snort: an overview of this popular Intrusion Detection System and its functionalities.

Module 3: Web Application Tools

This module examines tools used for scanning web applications for vulnerabilities, including:

  • Scanning for web vulnerabilities: understanding the importance of identifying security flaws.
  • Tools like Nikto and W3af for scanning web applications.
  • HTTP utilities: practical use of tools such as Curl, OpenSSL, and Stunnel.
  • Application Inspection tools: leveraging Zed Attack Proxy and Sqlmap for in-depth analysis.
  • Vulnerable Web Applications: hands-on experience with DVWA and Webgoat.
  • Password Cracking and Brute-Force Tools: using John the Ripper, L0htcrack, Pwdump, and HTC-Hydra to test password strength.

Module 4: Introduction to Cyber Crime and Law

This module provides insights into cybercrime, including:

  • Understanding Cyber Crimes: defining what constitutes a cybercrime and its various forms.
  • Types of Cybercrime: exploring different categories, including hacking and attack vectors.
  • Cyberspace and Criminal Behavior: examining the relationship between technology and criminal activity.
  • Clarification of Terms: defining key terms related to cybercrime.
  • Traditional Problems Associated with Computer Crime: discussing long-standing issues in the realm of cybersecurity.
  • Introduction to Incident Response and Digital Forensics: understanding how to respond to cyber incidents effectively.
  • Realms of the Cyber World: exploring different dimensions and implications of cyberspace.
  • Recognizing and Defining Computer Crime: identifying and classifying various cybercriminal activities.
  • Contemporary Crimes: analyzing modern crimes such as data contamination and destruction.
  • Indian IT ACT 2000: understanding the legal framework governing cyber activities in India.

Module 5: Introduction to Cyber Crime Investigation

This module covers the key concepts and tools used in cyber crime investigations, including:

  • Understanding Keyloggers and Spyware: how these tools compromise security.
  • Virus and Worms: analyzing malicious software and their impacts.
  • Trojans and Backdoors: recognizing threats posed by hidden entry points.
  • Steganography: techniques for hiding information within other files.
  • DOS and DDoS Attacks: understanding denial-of-service attacks and their prevention.
  • SQL Injection: examining vulnerabilities in web applications and database security.
  • Buffer Overflow: identifying and mitigating risks associated with buffer overflow attacks.
  • Attacks on Wireless Networks: understanding vulnerabilities specific to wireless networks.
