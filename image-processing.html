Image Processing

Module 1: Digital Image Fundamentals

This module introduces the foundational concepts of digital images, including:

  • Light and the electromagnetic spectrum.
  • Components of an image processing system.
  • Image formation and digitization concepts.
  • Neighbors of pixel adjacency and connectivity.
  • Regions and boundaries, distance measures.
  • Applications of digital image processing.

Module 2: Image Enhancements

This module covers techniques for enhancing images in both the spatial and frequency domains:

  • In Spatial Domain:
    • Basic gray level transformations.
    • Histogram processing.
    • Using arithmetic/logic operations.
    • Smoothing spatial filters.
    • Sharpening spatial filters.
  • In Frequency Domain:
    • Introduction to the Fourier transform and frequency domain concepts.
    • Smoothing frequency-domain filters.
    • Sharpening frequency domain filters.

Module 3: Image Restoration

This module discusses various approaches for restoring images:

  • Various noise models.
  • Image restoration using spatial domain filtering.
  • Image restoration using frequency domain filtering.
  • Estimating the degradation function.
  • Inverse filtering techniques.

Module 4: Colour Image Processing

This module focuses on the processing of color images, including:

  • Color fundamentals and models.
  • Color transformation techniques.
  • Smoothing and sharpening color images.
  • Color segmentation methods.

Module 5: Wavelet and Multi-resolution Processing

This module explores techniques for multi-resolution image processing:

  • Image pyramids.
  • Multi-resolution expansion.
  • Wavelet transform concepts.

Module 6: Image Compression

This module introduces concepts related to image compression:

  • Introduction to image compression models.
  • Error-free compression methods.
  • Lossy compression techniques.

Module 7: Image Segmentation

This module covers methods for segmenting images, including:

  • Detection of discontinuities.
  • Edge linking and boundary detection.
  • Thresholding techniques.
