Module 2: Problem Solving

This module delves into problem-solving methodologies in AI, including:

  • Problem-solving through searching: techniques and strategies to find solutions in AI.
  • State-space search techniques: an overview of various search algorithms, including:
    • Breadth-first search: exploring all nodes at the present depth before moving on to nodes at the next depth level.
    • Depth-first search: exploring as far as possible along each branch before backtracking.
    • Iterative deepening DFS: combining the space efficiency of depth-first search with the optimality of breadth-first search.
  • Heuristic search techniques: employing heuristics to improve the efficiency of search algorithms, including:
    • Hill climbing: a local search algorithm that continuously moves towards the direction of increasing value.
    • Best-first search: exploring paths based on the best available option according to a specified criterion.
    • Problem reduction: breaking down problems into simpler subproblems for easier management.
    • Constraint satisfaction: solving problems by finding values that meet specific constraints.

Module 3: Finding Optimal Paths

This module focuses on algorithms for finding optimal paths in various problem domains, including:

  • Brute force: exploring all possible solutions to find the optimal one.
  • Branch and bound: an optimization technique that divides problems into smaller subproblems and eliminates suboptimal solutions.
  • Algorithm A*: a widely-used algorithm for pathfinding and graph traversal that uses heuristics to improve efficiency.
  • Admissibility of A*: understanding the conditions under which the A* algorithm guarantees an optimal solution.
  • Iterative Deepening A*: combining the advantages of A* with iterative deepening techniques for enhanced performance.
  • Recursive best-first search: an efficient method for memory management in search algorithms.

Module 4: Planning

This module discusses planning techniques in AI systems, including:

  • Components of a planning system: understanding the key elements that contribute to effective planning.
  • Forward state space planning: planning from the initial state towards the goal state.
  • Backward state space planning: planning from the goal state back to the initial state.
  • Goal stack planning: managing goals and subgoals in a stack structure.
  • Plan space planning: exploring different possible plans simultaneously to find the most effective one.
  • Hierarchical planning: organizing plans into a hierarchy to simplify complex tasks.

Module 5: Game Tree Search

This module covers techniques for searching game trees in AI applications, including:

  • Minimax algorithm: a decision-making algorithm used in two-player games to minimize the possible loss for a worst-case scenario.
  • Alpha-beta pruning: an optimization technique for the minimax algorithm that reduces the number of nodes evaluated.
  • Heuristics in game tree search: the use of heuristics to enhance decision-making in complex games.
